----------------Windows快捷键-----------------------------------------

Alt+Tab在打开的应用不同窗口间进行切换

Windows+D：显示桌面或恢复桌面

Windows按住不动+Tab：打开的应用程序切换，立体效果（win7系统支持）

Ctrl+N 新建应用程序窗口

Alt+空格+C 关闭窗口

Alt+空格+N 最小化当前窗口

Alt+空格+X 最大化当前窗口

END 显示当前窗口的底端。

Home 显示当前窗口的顶端。

Ctrl+Home 光标快速移到文件头

Ctrl+End 光标快速移到文件尾

F2 重命名

-------------------vscode快捷键----------------------------

Ctrl + \	创建多个编辑器	【重要】抄代码利器

Ctrl + Enter	在当前行的下方新增一行，然后跳至该行	即使光标不在行尾，也能快速向下插入一行

Ctrl+Shift+Enter	在当前行的上方新增一行，然后跳至该行	即使光标不在行尾，也能快速向上插入一行

Alt + ↑	将代码向上移动	很常用

Alt + ↓	将代码向下移动	很常用

Alt + Shift + ↑	将代码向上复制	
Alt + Shift + ↓	将代码向下复制	写重复代码的利器

Alt + shift + F	代码格式化	很常用

Ctrl + Shift +F	全局搜索代码

Ctrl + P	在当前的项目工程里，全局搜索文件名
